# **App Name**: ProfitPilot

## Core Features:

- Dynamic Pricing Engine: Calculates the optimal platform price (Pp) for each product based on ingredient costs, fixed costs, desired profit margin, and DiDi's commission structure.
- Real-time Profitability Dashboard: Provides a visual dashboard displaying key financial KPIs such as net margin, cost performance index (CPI), and break-even point, allowing for quick assessment of profitability.
- Automated Promotion Scheduler: Enables users to schedule and manage promotional campaigns, calculating the impact on profit margins and ensuring promotions adhere to safe discount limits.
- AI-Powered Combo Suggestion Tool: Uses generative AI as a tool, specifically Gemini 2.5, to analyze sales data and suggest high-profitability product combinations (combos) that maximize revenue.
- Cost Variance Analysis: Track and analyze costs.
- Automated Margin Alerts: Generates notifications based on pre-configured thresholds, and can provide insight by way of generative AI.
- Task Calendar and Management: Displays an agenda including tasks to create a plan.
- Recipe Import from Excel: Página página dedicada a importar recetas desde un archivo Excel
- Ingredient Import from Excel: Página dedicada a importar ingredientes desde un archivo Excel
- Product Import from Excel: Página dedicada a importar productos desde un archivo Excel
- Sales Import from Excel: Página dedicada a importar ventas desde un archivo Excel
- Data Flow: Ingredients to Sales: Flujo de datos: ingredientes o inventario después recetas después productos después ventas
- Mental Map with React: Página dedicada a un mapa mental con react puede ser ortogonal,en círculo
- Specialized ETL Import with Excel: Feature; aplicación especializada en importación etl con Excel, normalización de datos, al importar un archivo preguntará cuál es el nombre de cada columna
- Combo Laboratory: Página dedicsda a laboratorio de combos a través de correspondencia matemática se crearán diversas todas las combinaciones posibles,combinaciones de primero segundo tercer y cuarto nivel de acuerdo a la correspondencia matemática
- Discount Simulator: Feature simulador de descuento e impacto de descuento sobre el margen deseado de ganancia

## Style Guidelines:

- Primary color: Saturated purple (#A64AC9), offering a sense of sophistication, quality, and innovation, with allusions to royalty that support the theme of 'maximizing' and 'optimizing' value.
- Background color: Light lavender (#E6E6FA), a pastel color from the same purple family that won't distract from the information.
- Accent color: Vibrant pink (#F39F5A), for buttons, highlights and key information; analogous to purple (a 'nearby' color) yet much more vibrant, ensuring high contrast to draw the eye.
- Headline font: 'Playfair', a modern serif font with high contrast and an elegant feel, to present information clearly.
- Body font: 'PT Sans', a humanist sans-serif that is easy to read and balances the style of Playfair. For text throughout the app.
- Code font: 'Source Code Pro' for displaying code snippets or formulas.
- Clean and modern icons that visually represent financial metrics and project management concepts. Consider using a consistent style with filled icons for emphasis.
- A structured layout with clear sections for financial data, project timelines, and AI recommendations. Utilize cards and tables to organize information efficiently.
- Subtle transitions and animations to provide feedback on user interactions, such as loading indicators, chart updates, and task completion markers.
